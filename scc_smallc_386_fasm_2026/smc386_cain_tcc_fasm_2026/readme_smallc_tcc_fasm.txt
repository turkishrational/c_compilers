'smallc1.exe' was created using tcc.

With the command 'tcc smallc1.c'.

'smallc1.asm' was created by running 'smallc1.exe' and using 'smallc1.c'
as the input file.

'smallcw.asm' (including 'smallc1.asm' without 'strlen') is used
to create a Win32 console program using fasm.

'fasm smallcw.asm SMALLCW.EXE'

'smallc2.asm' is the output file of 'SMALLCW.EXE'
using the input file 'smallc1.c'.

'SMALLC.PRG' was created using 'FASM' and 'smallc.asm'
(which includes the 'smallc1.asm' file).
